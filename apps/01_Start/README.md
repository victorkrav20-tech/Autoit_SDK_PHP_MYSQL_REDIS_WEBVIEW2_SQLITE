## 🎯 НАЗНАЧЕНИЕ

Приложение для чтения и отображения веб-страниц с использованием WebView2, MySQL и Redis.

---

## 🔧 АРХИТЕКТУРА

### Модульная структура:

**01_Start_Main.au3** - Главный файл
- Подключает модули Init и Events
- Запускает инициализацию
- Регистрирует обработчики событий
- Запускает основной цикл

**01_Start_Init.au3** - Модуль инициализации
- `_01_Start_Init()` - главная инициализация
  - Параметры: $bEnableMySQL, $bEnableRedis, $sRedisHost, $iRedisPort
  - Инициализирует SDK (Utils + Logger + Config)
  - Подключает MySQL (опционально)
  - Подключает Redis (опционально)
  - Инициализирует WebView2
  - Создаёт GUI
- `_01_Start_IsInitialized()` - проверка инициализации
- `_01_Start_IsMySQLEnabled()` - проверка доступности MySQL
- `_01_Start_IsRedisEnabled()` - проверка доступности Redis

**01_Start_Events.au3** - Модуль обработчиков событий
- `_01_Start_RegisterEvents()` - регистрация всех обработчиков
- `_01_Start_OnClose()` - обработчик закрытия приложения
- Заготовки будущих обработчиков:
  - `_01_Start_OnNavigate($sUrl)` - навигация по URL
  - `_01_Start_OnPageLoaded()` - загрузка страницы
  - `_01_Start_OnLoadError($sError)` - ошибка загрузки
  - `_01_Start_OnSaveHistory($sUrl, $sTitle)` - сохранение в историю
  - `_01_Start_OnAddBookmark($sUrl, $sTitle)` - добавление закладки

## 🚀 ИСПОЛЬЗОВАНИЕ

### Запуск приложения:
```autoit
; Запуск с MySQL и Redis
Local $bInit = _01_Start_Init(True, True, "127.0.0.1", 6379)

; Запуск только с MySQL
Local $bInit = _01_Start_Init(True, False)

; Запуск без баз данных
Local $bInit = _01_Start_Init(False, False)
```

### Проверка доступности модулей:
```autoit
If _01_Start_IsMySQLEnabled() Then
    ; Работа с MySQL
    _MySQL_Select("history", "*", "user_id=1")
EndIf

If _01_Start_IsRedisEnabled() Then
    ; Работа с Redis
    _Redis_Set("last_url", "https://example.com")
EndIf
```

## 📊 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Глобальные переменные:
- `$g_s01_Start_AppName` - имя приложения
- `$g_i01_Start_WindowID` - ID окна WebView2
- `$g_i01_Start_WindowWidth` - ширина окна (1200px)
- `$g_i01_Start_WindowHeight` - высота окна (700px)
- `$g_s01_Start_WindowTitle` - заголовок окна
- `$g_b01_Start_Initialized` - флаг инициализации
- `$g_b01_Start_MySQL_Enabled` - флаг доступности MySQL
- `$g_b01_Start_Redis_Enabled` - флаг доступности Redis

### Логирование:
Все логи пишутся в `logs/01_Start/01_Start_Main.log`

### Конфигурация:
- `config/app_config.json` - настройки окна и горячих клавиш
- `data/config.json` - настройки приложения (MySQL, Redis)
