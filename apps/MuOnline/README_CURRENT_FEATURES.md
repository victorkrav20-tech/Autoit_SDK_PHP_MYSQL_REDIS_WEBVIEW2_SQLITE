# 🎮 MU Online Bot - Текущие возможности

## ✅ Реализованные функции

### 1. 🪟 Работа с окнами игры
**Файл:** `Mu_Online_Main.au3`

**Функция:** `_Button_ReadWindows()`
- Сканирует все открытые окна MU Online
- Показывает список окон с именами персонажей
- Извлекает уровень персонажа из заголовка окна
- Позволяет выбрать окно для работы

**Как использовать:**
1. Запустить игру (одно или несколько окон)
2. Нажать кнопку "Прочитать окна"
3. Выбрать нужное окно из списка

---

### 2. 📸 Фоновый скриншот с OCR
**Файл:** `Mu_Online_Main.au3`

**Функция:** `_Button_Screenshot()`
- Создаёт фоновый скриншот окна (без активации)
- Распознаёт текст через Windows OCR (UWPOCR)
- Извлекает данные:
  - **Город** (название локации)
  - **Координаты** (X,Y позиция персонажа)
  - **Helper Bot** (активен/неактивен по цвету пикселя)

**Как использовать:**
1. Выбрать окно через "Прочитать окна"
2. Нажать кнопку "Скриншот"
3. Результаты отображаются в логе

**Технические детали:**
- Область OCR: верхний левый угол (настраиваемые координаты)
- Увеличение изображения в 4 раза для точности OCR
- Два режима: полный OCR или раздельный (город + координаты)
- Сохранение изображений в папку `img/`

### 3. 🗺️ Телепортация по городам
**Файл:** `Mu_Online_Send.au3`

**Функция:** `_Send_TeleportToCity($hWnd, $iY)`
- Открывает карту телепортации (клик по кнопке)
- Телепортирует в выбранный город (Ctrl+Click)
- Работает полностью в фоновом режиме (без активации окна)

**Координаты городов (Y):**
- `533` - Первый город в списке
- `551` - Второй город (+18px)
- `569` - Третий город (+18px)

**Как использовать:**
```autoit
_Send_TeleportToCity($hWnd, 533)  ; Телепорт в первый город
```

**Технические детали:**
- Кнопка открытия карты: `(686, 840)`
- Координата X для клика: `200` (100px левее центра)
- Использует `PostMessage` для фоновых кликов
- Карта закрывается автоматически после телепорта

---

### 4. 🖱️ Фоновые клики
**Файл:** `Mu_Online_Send.au3`

**Функция:** `_Send_Click($hWnd, $iX, $iY, $bCtrl = False)`
- Кликает в указанную точку окна
- Поддерживает Ctrl+Click
- Работает в фоновом режиме через `PostMessage`
- Не активирует окно игры

**Как использовать:**
```autoit
_Send_Click($hWnd, 400, 300)        ; Обычный клик
_Send_Click($hWnd, 400, 300, True)  ; Ctrl+Click
```

**Технические детали:**
- Использует `WM_MOUSEMOVE`, `WM_LBUTTONDOWN`, `WM_LBUTTONUP`
- Задержки между событиями: 50ms
- Координаты относительно окна

### 5. 🎯 Система маяков (маркеров)
**Файл:** `Mu_Online_Markers.au3`

**Функции:**
- `_Markers_Init()` - инициализация системы
- `_Markers_Load()` - загрузка маркеров из JSON
- `_Markers_Save()` - сохранение маркеров
- `_Markers_ScanAll()` - сканирование всех маркеров
- `_Markers_Check($sName)` - проверка конкретного маркера
- `_Markers_GetPixelColor($hWnd, $iX, $iY)` - получение цвета пикселя

**Файл конфигурации:** `data/markers.json`

**Структура маркера:**
```json
{
  "name": "helper_active",
  "x": 322,
  "y": 35,
  "color": "0xFFFF00",
  "description": "Проверка активности Helper Bot"
}
```

**Как использовать:**
1. Добавить маркеры в `data/markers.json`
2. Нажать кнопку "Сканировать маяки"
3. Система проверит цвета пикселей и покажет результат

**Технические детали:**
- Гибридная система: первое сканирование заполняет цвета, последующие сравнивают
- Координаты относительно окна
- Фоновое чтение пикселей через `_CaptureWindowBackground()`

### 6. 🔧 Вспомогательные функции
**Файл:** `Mu_Online_Core.au3`

**Функции:**
- `_Core_GetGameData($hWnd)` - получение данных игры (город, координаты, helper)
