## 📋 О ДОКУМЕНТЕ

**Назначение:** Архив завершённых задач для сохранения истории без раздувания ROADMAP.md

**Структура:**
- Первые 30 строк - метаинформация и инструкция по работе
- После 30 строк - завершённые задачи (от новых к старым)

**Как работать с документом:**
1. При завершении задачи из ROADMAP.md переносить её сюда
2. Новые записи вставлять ПОСЛЕ этого блока (строка 31+) и ПЕРЕД предыдущими записями
3. Обновлять "Последнее обновление" и "Текущая версия SDK" в шапке
4. Группировать по версиям SDK → модулям → датам
5. Указывать что принесло обновление, где проверено, результаты тестов

**Формат записи:**

**Структура SQLite таблицы queue:**
```sql
CREATE TABLE queue (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    uuid TEXT UNIQUE NOT NULL,
    server TEXT NOT NULL,
    sql_query TEXT NOT NULL,
    params TEXT,
    is_json INTEGER DEFAULT 0,
    retry_count INTEGER DEFAULT 0,
    created_at TEXT NOT NULL,
    last_attempt TEXT,
    status TEXT DEFAULT 'pending'
);
CREATE INDEX idx_queue_server_status ON queue(server, status);
CREATE INDEX idx_queue_created ON queue(created_at);
CREATE INDEX idx_queue_uuid ON queue(uuid);
```
